# Cheatsheets - Windows Command Shell
Search the entire filesystem for a file. 
```bash
dir "\network-secret.txt" /s
```

Get the kernel version.
```bash
systeminfo | findstr Build

# output
OS Version:                10.0.17763 N/A Build 17763
```
